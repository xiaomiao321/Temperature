# RS485 通讯协议 (Modbus RTU)

## 1. 默认参数

| 参数 | 值 |
|------|-----|
| 设备地址 | 1 |
| 波特率 | 115200 |
| 数据位 | 8 |
| 停止位 | 1 |
| 奇偶校验 | NONE |

## 2. 通讯帧格式

```
Addr | Cmd | Data(n 字节) | Crc(2 字节)
```

## 3. 支持的功能码

| Cmd | 含义 | 备注 |
|-----|------|------|
| 0x04 | 读输入寄存器 | PT100 温度 |
| 0x03 | 读保持寄存器 | 设备参数或 PT100 温度 |
| 0x06 | 写单个保持寄存器 | 写设备参数 |
| 0x10 | 写多个保持寄存器 | 写设备参数 |

---

## 4. 温度值读取

设备提供 2 种格式的温度值输出：
- **16 位有符号整形温度值**（放大 10 倍）
- **32 位 IEEE-754 浮点数格式温度值**

### 4.1 16 位有符号整形温度值

通过输入寄存器命令 `0x04` 或读保持寄存器命令 `0x03` 读取。

**数据格式**：每个寄存器值表示 1 路温度值（放大 10 倍后），采用 16 位有符号格式。

| 寄存器值 | 十进制 | 实际温度值 |
|---------|--------|-----------|
| 0x0131 | 305 | 30.5℃ |
| 0xFECF | -305 | -30.5℃ |
| 0xFF89 | -119 | -11.9℃ |

> **特殊值**：当寄存器值为 `0xEC78` (十进制 -5000) 时，表示传感器未接入或线路故障。

#### 寄存器地址表

| 寄存器地址 | 含义 |
|-----------|------|
| 0x0000 | 第 1 路 PT100 温度（放大 10 倍） |
| 0x0001 | 第 2 路 PT100 温度（放大 10 倍） |
| 0x0002 | 第 3 路 PT100 温度（放大 10 倍） |
| 0x0003 | 第 4 路 PT100 温度（放大 10 倍） |
| 0x0004 | 第 5 路 PT100 温度（放大 10 倍） |
| 0x0005 | 第 6 路 PT100 温度（放大 10 倍） |
| 0x0006 | 第 7 路 PT100 温度（放大 10 倍） |
| 0x0007 | 第 8 路 PT100 温度（放大 10 倍） |

#### 读取示例（设备地址 1，读取前 8 路温度）

**使用 0x04 命令（读输入寄存器）**：
```
发送：01 04 00 00 00 08 F1 CC
返回：01 04 10 00 00 00 00 FF FF 00 00 00 00 00 00 00 00 00 00 31 2E
```

**使用 0x03 命令（读保持寄存器）**：
```
发送：01 03 00 00 00 08 44 0C
返回：01 03 10 FF FF EC 78 00 01 00 01 EC 78 00 02 00 02 00 04 31 48
```

---

### 4.2 32 位 IEEE-754 浮点数格式温度值

通过输入寄存器命令 `0x04` 或读保持寄存器命令 `0x03` 读取。

**数据格式**：每 2 个寄存器值表示 1 路温度值，采用 32 位 IEEE-754 浮点数格式。

#### 寄存器地址表

| 寄存器地址 | 通道号 | 备注 |
|-----------|--------|------|
| 0x0020 | 1 | 数据格式符合 32 位 IEEE-754 浮点数格式 |
| 0x0022 | 2 | 方便与其他标准设备对接 |
| 0x0024 | 3 | |
| 0x0026 | 4 | |
| 0x0028 | 5 | |
| 0x002A | 6 | |
| 0x002C | 7 | |
| 0x002E | 8 | |

#### 读取示例（设备地址 1，读取前 8 路温度）

**使用 0x04 命令（读输入寄存器）**：
```
发送：01 04 00 20 00 10 F0 0C
返回：01 04 20 00 00 00 00 00 00 00 00 CC CD 3D CC CC CD 3D CC 00 00 C3 FA 00 00 C3 FA 00 00 C3 FA 00 00 C3 FA B3 69
```

**使用 0x03 命令（读保持寄存器）**：
```
发送：01 03 00 20 00 10 45 CC
返回：01 03 20 00 00 00 00 00 00 C3 FA CC CD 3D CC CC CD 3D CC 00 00 C3 FA CC CD 3E 4C CC CD 3E 4C CC CD 3E CC BD 27
```

## 5. 设备参数配置

通过读写保持寄存器可以实现对设备参数的读取和写入，参数保存后掉电不丢失。

> **注意**：与 CAN(FD) 相关的寄存器仅 PT100D22、PT100D42、PT100D82、PT100B82 支持。

### 保持寄存器地址表

| 地址 | 名称 | 复位值 | 含义/取值范围 |
|------|------|--------|--------------|
| 0x0100 | 设备地址 | 1 | 0~255 |
| 0x0101 | 自动上报周期 | 0 | 单位：ms；0=禁止上报；最小上报周期 100ms |
| 0x0102 | 串口波特率 | 1152 | 实际波特率÷100（如：12=1200，96=9600，1152=115200，10240=1024000） |
| 0x0103 | 串口数据位 | 8 | 支持 7/8/9 |
| 0x0104 | 串口奇偶校验 | 0x4E | 0x4E=不校验；0x45=偶校验；0x4F=奇校验 |
| 0x0105 | 串口停止位 | 1 | 1=1bit；2=1.5bit；3=2bit |
| 0x0106 | CAN 类型 | 1 | 0=CAN；1=CANFD |
| 0x0107 | CANFD 加速 | 0 | 0=不加速；1=加速 |
| 0x0108 | CAN(FD) 自定义波特率 | 0 | 0=常用波特率码；1=自定义波特率 |
| 0x0109 | CAN(FD) 波特率码 | 0x45 | 低字节有效：高 4bit=仲裁域波特率码，低 4bit=数据域波特率码 |
| 0x010A | 仲裁域 SJW | 0 | 仲裁域波特率同步跳转宽度 |
| 0x010B | 仲裁域 TSEG1 | 12 | 仲裁域波特率时间段 1 |
| 0x010C | 仲裁域 TSEG2 | 1 | 仲裁域波特率时间段 2 |
| 0x010D | 仲裁域 BRP | 15 | 仲裁域波特率分频 |
| 0x010E | 数据域 SJW | 0 | 数据域波特率同步跳转宽度 |
| 0x010F | 数据域 TSEG1 | 11 | 数据域波特率时间段 1 |
| 0x0110 | 数据域 TSEG2 | 1 | 数据域波特率时间段 2 |
| 0x0111 | 数据域 BRP | 8 | 数据域波特率分频 |
| 0x0112 | 帧类型 | 0 | 0=标准帧；1=扩展帧 |
| 0x0113~0x011F | 系统预留 | 0 | - |
| 0x0120 | 恢复出厂 | 0 | 写 0x0001 恢复出厂设置 |
| 0x0121 | 复位 | 0 | 写 0x0001 复位 |
