## 硬件引脚配置

根据 CubeMX 配置 (`main.h`)：

| 信号 | 引脚 | 功能 |
|------|------|------|
| DS | PA5 | 串行数据输入 |
| SHCP | PA6 | 移位寄存器时钟 |
| STCP | PA7 | 输出寄存器时钟 (锁存) |

## 驱动 API 说明

### 初始化函数
```c
void HC595_Init(void);
// 初始化 GPIO 引脚 (CubeMX 已配置时可省略)
```

### 基本发送函数
```c
void HC595_SendByte(uint8_t data);
// 发送 1 字节数据到 74HC595

void HC595_Latch(void);
// 锁存数据到输出
```

### 显示函数
```c
void HC595_Display(uint8_t num);
// 显示单个数字 (0-15)，自动查表并锁存

void HC595_DisplayRaw(uint8_t seg_code);
// 显示原始段码值

void HC595_SendMultiBytes(uint8_t *data, uint8_t count);
// 发送多字节 (用于级联)

void HC595_Display8Digits(uint8_t *digits);
// 显示 8 位数码管 (8 片 74HC595 级联)
```

### 段码表
```c
extern const uint8_t SEG_CODE[];
// 共阳极段码表：0-9, A-F, 灭，点
// SEG_CODE[0] = 0xC0
// SEG_CODE[1] = 0xCF
// ...
// SEG_CODE[15] = 0x8E (F)
// SEG_CODE[16] = 0xFF (灭)
// SEG_CODE[17] = 0x7F (点)
```

## 使用示例

### 示例 1：循环显示 0-9 (已实现于 main.c)
```c
#include "hc595.h"

int main(void)
{
    HAL_Init();
    SystemClock_Config();
    MX_GPIO_Init();
    
    HC595_Init();  // 初始化 74HC595

    while(1)
    {
        for(uint8_t i = 0; i < 10; i++)
        {
            HC595_Display(i);
            HAL_Delay(500);
        }
    }
}
```

### 示例 2：显示固定数字
```c
// 显示数字 5
HC595_Display(5);

// 显示字母 A
HC595_Display(10);  // 10 对应 A

// 显示字母 F
HC595_Display(15);  // 15 对应 F
```

### 示例 3：8 位数码管级联显示
```c
// 显示 "12345678"
uint8_t digits[] = {1, 2, 3, 4, 5, 6, 7, 8};
HC595_Display8Digits(digits);
```

### 示例 4：自定义段码显示
```c
// 显示自定义图案
HC595_DisplayRaw(0xAA);  // 显示特定段组合
```

## 与共阳极/共阴极数码管的匹配

### 共阳极数码管 (默认)
- 段码表：`SEG_CODE[]` (0xC0, 0xCF, ...)
- 公共端接 VCC，段选低电平点亮

### 共阴极数码管
如需使用共阴极数码管，修改段码表取反即可：
```c
const uint8_t SEG_CODE_CATHODE[] = {
    0x3F,  /* 0 */
    0x30,  /* 1 */
    0x5B,  /* 2 */
    0x4F,  /* 3 */
    0x66,  /* 4 */
    0x6D,  /* 5 */
    0x7D,  /* 6 */
    0x07,  /* 7 */
    0x7F,  /* 8 */
    0x6F,  /* 9 */
    // ...
};
```

## 常见问题

### Q1: 数码管不亮
- 检查硬件连接 (DS/SHCP/STCP 引脚)
- 确认数码管类型 (共阳/共阴)
- 检查限流电阻是否合适

### Q2: 显示乱码
- 确认段码表与数码管类型匹配
- 检查引脚顺序是否正确

### Q3: 亮度不足
- 减小限流电阻阻值
- 检查电源供电电压
