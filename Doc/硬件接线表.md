# 硬件接线表

## 1. 数码管接线 (74HC595 模块)

### 第一组数码管 (主显示)

| 模块引脚 | STM32 引脚 | GPIO | 说明 |
|---------|-----------|------|------|
| **DS/SDI** | **PA5** | GPIO_PIN_5 | 串行数据输入 |
| **SHCP/SCLK** | **PA6** | GPIO_PIN_6 | 移位寄存器时钟 |
| **STCP/LOAD** | **PA7** | GPIO_PIN_7 | 输出寄存器时钟 (锁存) |
| **VCC** | 3.3V | - | 电源 |
| **GND** | GND | - | 地 |

### 第二组数码管 (扩展显示)

| 模块引脚 | STM32 引脚 | GPIO | 说明 |
|---------|-----------|------|------|
| **DS/SDI** | **PB7** | GPIO_PIN_7 | 串行数据输入 |
| **SHCP/SCLK** | **PB8** | GPIO_PIN_8 | 移位寄存器时钟 |
| **STCP/LOAD** | **PB9** | GPIO_PIN_9 | 输出寄存器时钟 (锁存) |
| **VCC** | 3.3V | - | 电源 |
| **GND** | GND | - | 地 |

## 2. 按键接线

| 按键 | 功能 | STM32 引脚 | GPIO | 接线方式 |
|-----|------|-----------|------|---------|
| **BTN1** | SET 确认 | **PB3** | GPIO_PIN_3 | 按键→GND |
| **BTN2** | 加 | **PB4** | GPIO_PIN_4 | 按键→GND |
| **BTN3** | 减 | **PB0** | GPIO_PIN_0 | 按键→GND |
| **BTN4** | 左移/切换 | **PB1** | GPIO_PIN_1 | 按键→GND |

**按键接法**：按键一端接 GPIO，另一端接 **GND**（低电平有效）

---

## 3. LED 指示灯接线

| 引脚 | STM32 引脚 | GPIO | 说明 |
|-----|-----------|------|------|
| **LED** | **PC13** | GPIO_PIN_13 | 超温指示灯 |

**LED 接法**：
- LED 正极 → 限流电阻 (220Ω~1kΩ) → **3.3V**
- LED 负极 → **PC13**
- **低电平点亮**

## 4. 串口接线

### UART1 (连接上位机/电脑)

| 功能 | STM32 引脚 | GPIO | 连接 |
|-----|-----------|------|------|
| **TX** | **PA9** | GPIO_PIN_9 | → 上位机 RX |
| **RX** | **PA10** | GPIO_PIN_10 | → 上位机 TX |
| **GND** | GND | - | 共地 |

### UART2 (连接温度模块)

| 功能 | STM32 引脚 | GPIO | 连接 |
|-----|-----------|------|------|
| **TX** | **PA2** | GPIO_PIN_2 | → 温度模块 RX |
| **RX** | **PA3** | GPIO_PIN_3 | → 温度模块 TX |
| **GND** | GND | - | 共地 |

**注意**：
- RS485 模块可能需要 MAX3232 电平转换
- 确保所有设备 **共地 (GND)**

## 8. 按键操作

### 正常模式

| 按键 | 功能 |
|-----|------|
| **SET (BTN1)** | 短按进入设置模式 |
| **移位 (BTN4)** | 无响应 |
| **加 (BTN2)** | 无响应 |
| **减 (BTN3)** | 无响应 |

### 设置模式

| 按键 | 功能 |
|-----|------|
| **SET (BTN1)** | 短按确认并退出设置 |
| **SET (BTN1)** | **长按 1 秒** 切换设置的数码管组 (第一组↔第二组) |
| **移位 (BTN4)** | 切换位数 (个位→十位→百位→个位循环) |
| **加 (BTN2)** | 当前位 +1 |
| **减 (BTN3)** | 当前位 -1 |

### 设置流程

1. 按 **SET** 进入设置模式（当前设置的组某位闪烁）
2. **长按 SET 键 1 秒** 切换设置第一组或第二组数码管
3. 按 **移位** 选择要修改的位
4. 按 **加/减** 修改数值
5. 按 **SET** 确认保存并退出

## 9. 温度通道分组

| 组别 | 数码管 | 温度通道 | 说明 |
|-----|--------|---------|------|
| **第一组** | 第一组数码管 (PA5-PA7) | CH1-CH4 | 独立设置阈值 1 |
| **第二组** | 第二组数码管 (PB7-PB9) | CH5-CH8 | 独立设置阈值 2 |

**超温报警逻辑**：
- CH1-CH4 超温 → 使用阈值 1 判断 → LED 点亮
- CH5-CH8 超温 → 使用阈值 2 判断 → LED 点亮
- 任意通道超温都会触发报警

## 9. 调试变量

在调试器中可查看以下变量：

| 变量名 | 说明 |
|-------|------|
| `DBG_setValue` | 当前设置值 (0-999) |
| `DBG_setMode` | 0=正常，1=设置 |
| `DBG_setPos` | 0=百位，1=十位，2=个位 |
| `DBG_blinkState` | 闪烁状态 |
| `DBG_seg_d1/d2/d3` | 显示值 (百/十/个位) |

## 10. 常见问题

| 问题 | 可能原因 | 解决方法 |
|-----|---------|---------|
| 数码管不亮 | 电源/接线错误 | 检查 VCC/GND/信号线 |
| 显示乱码 | 段码表不匹配 | 确认共阳/共阴极 |
| 按键无响应 | 接线错误 | 检查按键是否接 GND |
| LED 不亮 | 极性接反 | 调换 LED 正负极 |
