# 数码管显示方案设计方案 (V4)

## 1. 硬件配置

| 数码管组 | 位数 | 控制引脚 | 用途 |
|---------|------|---------|------|
| 第一组 (HC595-1) | 3 位 | PA5-PA7 | 显示 CH1-CH7 相关数据 |
| 第二组 (HC595-2) | 3 位 | PB7-PB9 | 显示 CH8 相关数据 |

---

## 2. 显示模式总览

| 模式 | 名称 | 触发条件 | 说明 |
|------|------|---------|------|
| M0 | 轮询显示 | 默认/上电/退出设置/无操作 | 轮流显示各通道通道号和温度 |
| M1 | 报警优先 | 有通道超温 | 强制显示报警通道信息 |
| M2 | 设置模式 | 按 SET 键 | 修改阈值 |

---

## 3. 上电自检序列

**触发条件**：每次上电/复位时执行一次

**自检流程**：
```
步骤  →  第一组显示  →  第二组显示  →  持续时间
─────────────────────────────────────────────
1        [  8  8  8]     [  8  8  8]     500ms  (全段测试)
2        [  -  -  -]     [  -  -  -]     200ms  (全灭)
3        [  1  2  3]     [  4  5  6]     500ms  (数字测试)
4        [  -  -  -]     [  -  -  -]     200ms  (全灭)
5        [  0  3  0]     [  0  3  0]     300ms  (默认阈值)
─────────────────────────────────────────────
自检完成，进入轮询显示模式
```

## 4. 模式 M0：轮询显示（默认模式）

### 4.1 轮询逻辑

**触发条件**：
- 上电自检完成后进入（默认模式）
- 退出设置模式后进入
- 报警解除后返回

**显示节奏**：
```
每个通道分 2 步显示：
  步骤 1: 显示通道号 (1.5 秒)  → 格式：CHn
  步骤 2: 显示温度值 (1.5 秒)  → 格式：温度数值
  ─────────────────────────────
  每通道总计 3 秒，8 通道循环一遍共 24 秒
```

### 4.2 显示顺序

```
周期   →  第一组数码管  →  第二组数码管  →  持续时间
─────────────────────────────────────────────────
CH1     [  C  H  1]     [  C  H  8]     1.5 秒   (通道号)
        [  2  5• 3]     [  2  4• 3]     1.5 秒   (温度值)
CH2     [  C  H  2]     [  C  H  8]     1.5 秒
        [  2  3• 6]     [  2  4• 3]     1.5 秒
CH3     [  C  H  3]     [  C  H  8]     1.5 秒
        [  2  4• 1]     [  2  4• 3]     1.5 秒
CH4     [  C  H  4]     [  C  H  8]     1.5 秒
        [  E  R  R]     [  2  4• 3]     1.5 秒   (CH4 无传感器)
CH5     [  C  H  5]     [  C  H  8]     1.5 秒
        [  E  R  R]     [  2  4• 3]     1.5 秒   (CH5 无传感器)
CH6     [  C  H  6]     [  C  H  8]     1.5 秒
        [  E  R  R]     [  2  4• 3]     1.5 秒   (CH6 无传感器)
CH7     [  C  H  7]     [  C  H  8]     1.5 秒
        [  E  R  R]     [  2  4• 3]     1.5 秒   (CH7 无传感器)
CH8     [  C  H  8]     [  C  H  8]     1.5 秒
        [  2  4• 3]     [  2  4• 3]     1.5 秒   (CH8 温度)
─────────────────────────────────────────────────
循环回到 CH1
```

### 4.3 通道号显示格式

**两组数码管显示规则相同**：
```
格式：[  C  H  n]  → CH1~CH8 的通道号

第一组 (CH1-CH7):
  CH1: [  C  H  1]
  CH2: [  C  H  2]
  CH3: [  C  H  3]
  CH4: [  C  H  4]
  CH5: [  C  H  5]
  CH6: [  C  H  6]
  CH7: [  C  H  7]

第二组 (CH8 固定):
  始终显示：[  C  H  8]
```

### 4.4 温度值显示格式

**两组数码管显示规则相同**：
```
正常温度 (0℃ ~ 99.9℃):
  格式：[  X  Y• Z]  → XY.Z℃
        │  │  └─ 个位 (小数部分×10)
        │  └──── 十位 (整数部分个位) + 小数点
        └──────── 百位 (整数部分十位)
  
  示例：25.3℃ → [  2  5• 3]

≥100℃:
  格式：[  X  Y  Z]  → 只显示整数部分，无小数点
        │  │  └─ 个位
        │  └──── 十位
        └──────── 百位
  
  示例：125℃ → [  1  2  5]

<0℃:
  格式：[  -  X  Y]  → -XY.Z℃
        │  │  └─ 整数部分个位
        │  └──── 整数部分十位
        └──────── 负号
  
  示例：-15.3℃ → [  -  1  5]
```

### 4.5 特殊情况处理

**两组数码管处理规则相同**：

| 情况 | 显示格式 | 示例 | 说明 |
|------|---------|------|------|
| **传感器故障** | `[  E  R  R]` | CH4 无传感器 | 显示 ERR |
| **温度 ≥100℃** | `[  1  2  5]` | 125.6℃ | 只显示整数部分，无小数点 |
| **温度 ≥1000℃** | `[  H  i  G]` | 超量程 | 显示 HiG (HIGH) |
| **温度 <0℃** | `[  -  1  5]` | -15.3℃ | 百位显示负号，后两位显示绝对值 |
| **温度 <-99.9℃** | `[  L  o  -]` | 超低温 | 显示 Lo- (LOW) |

### 4.6 温度显示示例

**正常温度显示**：
```
CH1 = 25.3℃:
  通道号：第一组 [  C  H  1]  第二组 [  C  H  8]
  温度值：第一组 [  2  5• 3]  第二组 [  2  4• 3]

CH8 = 36.2℃:
  通道号：第一组 [  C  H  8]  第二组 [  C  H  8]
  温度值：第一组 [  2  4• 3]  第二组 [  3  6• 2]
```

**高温显示**：
```
CH3 = 125.6℃:
  通道号：第一组 [  C  H  3]  第二组 [  C  H  8]
  温度值：第一组 [  1  2  5]  第二组 [  2  4• 3]  (无小数点)
```

**低温显示**：
```
CH2 = -15.3℃:
  通道号：第一组 [  C  H  2]  第二组 [  C  H  8]
  温度值：第一组 [  -  1  5]  第二组 [  2  4• 3]  (百位为负号)
```

**传感器故障显示**：
```
CH4 未接线:
  通道号：第一组 [  C  H  4]  第二组 [  C  H  8]
  温度值：第一组 [  E  R  R]  第二组 [  2  4• 3]
```

## 5. 模式 M1：报警优先显示

### 5.1 触发条件

- 任意通道温度 ≥ 对应阈值（CH1-CH7 用阈值 1，CH8 用阈值 2）
- 手动报警按钮按下

### 5.2 报警显示流程

**单个通道报警时**：
```
每 3 秒一个周期，循环显示：
  步骤 1: 报警标识 [  A  L  A]  →  1 秒
  步骤 2: 报警通道号 [  C  H  n]  →  1 秒
  步骤 3: 报警温度值 [  X  X• X]  →  1 秒
  ───────────────────────────────────────
  总计 3 秒，然后重复
```

**多个通道报警时**：
```
依次轮流显示每个报警通道：

CH3 和 CH8 同时超温时：
  周期 1 (CH3):
    [  A  L  A]  [  C  H  8]   1 秒  (报警标识)
    [  C  H  3]  [  C  H  8]   1 秒  (CH3 通道号)
    [  3  2• 5]  [  2  4• 3]   1 秒  (CH3 温度 32.5℃)
  
  周期 2 (CH8):
    [  A  L  A]  [  C  H  8]   1 秒  (报警标识)
    [  C  H  8]  [  C  H  8]   1 秒  (CH8 通道号)
    [  2  4• 3]  [  3  6• 2]   1 秒  (CH8 温度 36.2℃)
  
  ───────────────────────────────────────
  循环回到周期 1
```

### 5.3 报警显示示例

**场景 1：CH3 超温（阈值 1=30℃，实际 32.5℃）**
```
每 3 秒循环：
  [  A  L  A]  [  C  H  8]   → 报警标识 (1 秒)
  [  C  H  3]  [  C  H  8]   → CH3 通道号 (1 秒)
  [  3  2• 5]  [  2  4• 3]   → CH3 温度 32.5℃ (1 秒)
```

**场景 2：CH8 超温（阈值 2=35℃，实际 36.2℃）**
```
每 3 秒循环：
  [  A  L  A]  [  C  H  8]   → 报警标识 (1 秒)
  [  C  H  8]  [  C  H  8]   → CH8 通道号 (1 秒)
  [  2  4• 3]  [  3  6• 2]   → CH8 温度 36.2℃ (1 秒)
```

**场景 3：多通道报警（CH2、CH3、CH8 同时超温）**
```
每 9 秒一个大周期：

第 1-3 秒 (CH2):
  [  A  L  A]  [  C  H  8]   → 报警标识
  [  C  H  2]  [  C  H  8]   → CH2 通道号
  [  3  1• 2]  [  2  4• 3]   → CH2 温度 31.2℃

第 4-6 秒 (CH3):
  [  A  L  A]  [  C  H  8]   → 报警标识
  [  C  H  3]  [  C  H  8]   → CH3 通道号
  [  3  2• 5]  [  2  4• 3]   → CH3 温度 32.5℃

第 7-9 秒 (CH8):
  [  A  L  A]  [  C  H  8]   → 报警标识
  [  C  H  8]  [  C  H  8]   → CH8 通道号
  [  2  4• 3]  [  3  6• 2]   → CH8 温度 36.2℃

──────────────────────────────
循环回到第 1 秒
```

**场景 4：手动报警按钮按下**
```
每 3 秒循环：
  [  A  L  A]  [  C  H  8]   → 报警标识 (1 秒)
  [  M  A  N]  [  C  H  8]   → 手动报警 (1 秒)
  [  C  H  8]  [  C  H  8]   → CH8 通道号 (1 秒)
```

### 5.4 报警闪烁规则

- 多通道报警时，按通道号顺序轮流显示
- 报警优先于轮询显示，打断 M0 模式
- 每个报警通道显示 3 秒（标识 1 秒 + 通道号 1 秒 + 温度 1 秒）

### 5.5 报警解除

- 温度降到阈值以下 + 按复位键 → 报警解除
- 报警解除后返回轮询显示模式（M0）

## 6. 模式 M2：设置模式

### 6.1 触发条件

- 按 SET 键进入

### 6.2 设置内容

```
设置项  →  第一组显示  →  第二组显示
─────────────────────────────
阈值 1     [  0  3  0]   [  -  -  - ]
阈值 2     [  -  -  - ]   [  0  3  5]
─────────────────────────────
```

### 6.3 设置显示规则

- **不显示小数点**：阈值 30℃ 显示为 `030`
- **百位为 0 时显示**：与轮询模式不同，设置模式下百位的 0 要显示
- **组指示**：通过第一组/第二组分别显示来指示当前设置的组

### 6.4 设置操作

| 按键 | 功能 |
|------|------|
| SET | 确认/切换下一项/退出 |
| SHIFT | 选择位数（百位/十位/个位） |
| + | 数值增加 |
| - | 数值减少 |
| 长按 SET(1 秒) | 切换组（阈值 1↔阈值 2） |

### 6.5 退出设置

- 按 SET 键确认退出
- 退出后直接进入轮询显示模式（M0）

## 10. 段码表

```c
/* 共阳极段码表：0-9, A-F, 特殊字符 */
/* 段顺序：a,b,c,d,e,f,g,dp (dp 为小数点) */
/* 共阳极：0 点亮，1 熄灭 */
const uint8_t SEG_CODE[] = {
    0xC0,  /* 0: a,b,c,d,e,f 亮 */
    0xF9,  /* 1: b,c 亮 */
    0xA4,  /* 2: a,b,d,e,g 亮 */
    0xB0,  /* 3: a,b,c,d,g 亮 */
    0x99,  /* 4: b,c,f,g 亮 */
    0x92,  /* 5: a,c,d,f,g 亮 */
    0x82,  /* 6: a,c,d,e,f,g 亮 */
    0xF8,  /* 7: a,b,c 亮 */
    0x80,  /* 8: 全亮 */
    0x90,  /* 9: a,b,c,d,f,g 亮 */
    0x88,  /* A: a,b,c,e,f,g 亮 */
    0x83,  /* b: c,d,e,f,g 亮 (小写 b) */
    0xC6,  /* C: a,d,e,f 亮 */
    0xA1,  /* d: b,c,d,e,f 亮 (小写 d) */
    0x86,  /* E: a,d,e,f,g 亮 */
    0x8E,  /* F: a,e,f,g 亮 */
    0x89,  /* H: b,c,e,f,g 亮 */
    0xC1,  /* L: d,e,f 亮 */
    0xA3,  /* r: e,f,g 亮 (小写 r) */
    0x91,  /* n: c,e,g 亮 (小写 n) */
    0x87,  /* y: b,c,d,f,g 亮 */
    0xBF,  /* -: 只有 g 段亮 (负号) */
    0xFF,  /* 灭：全灭 */
};

/* 小数点掩码：与段码进行 AND 操作可添加小数点 */
#define SEG_DOT_MASK  0x7F  /* 最高位置 0，点亮小数点 */

/* 特殊显示宏 */
#define SEG_DOT(code)     ((code) & SEG_DOT_MASK)  /* 添加小数点 */
#define SEG_MINUS         0xBF                     /* 负号 */
#define SEG_OFF           0xFF                     /* 熄灭 */
```

## 11. 实现优先级

| 优先级 | 功能 | 预计工时 |
|-------|------|---------|
| P0 | 轮询显示（通道号 + 温度） | 1.5h |
| P0 | 报警优先显示 | 1h |
| P0 | 特殊字符显示（ERR, -, HiG） | 0.5h |
| P1 | 小数点显示控制 | 0.5h |
| P1 | 多通道报警轮流显示 | 0.5h |
| P2 | 设置模式显示（无小数点） | 0.5h |
| P2 | 上电自检序列优化 | 0.5h |

## 12. 备注

1. **通道号显示**：两组数码管都显示 `CHn` 格式，第一组 CH1-CH7，第二组固定 CH8
2. **温度显示**：两组数码管显示对应通道的温度值
3. **报警显示**：每 3 秒循环显示：报警标识 (ALA) → 通道号 (CHn) → 温度值
4. **高温处理**：≥100℃时只显示整数部分，不显示小数点
5. **低温处理**：<0℃时百位显示负号，后两位显示绝对值
6. **故障显示**：传感器故障显示 `ERR`
7. **小数点**：
   - 轮询模式：正常温度时十位和个位之间显示小数点
   - 设置模式：不显示小数点，30℃显示为 `030`
8. **无操作**：保持轮询显示，不进入省电模式
